Variance swap

A variance swap is an instrument which allows investors to trade future realized (or historical) volatility against current implied volatility. Only variance the squared volatility can be replicated with a static hedge.

Index Amortising Swap

Swap Buyer receives the following coupons: Buyer rate paid e.g. quarterly (according to specified day count convention) on the principal amount, amortised according to the values of the buyer rate

Swap Seller receives seller rate on same coupon dates.
